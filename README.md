# Mateřská Odpočet MVP (Mobile-First)

Tato aplikace slouží jako odpočítávání do odchodu na mateřskou (13. 1. 2026 – 10. 3. 2026).

## Rychlý start

1. Nainstalujte závislosti:
   ```bash
   npm i
   ```

2. Spusťte vývojový server:
   ```bash
   npm run dev
   ```

3. Sestavte produkční verzi:
   ```bash
   npm run build
   ```

## Funkce
- Mobile-first dark UI.
- Logika odškrtávání pouze aktuálního dne.
- Insider korporátní hantýrka (Alza style).
- Ukládání stavu do localStorage.
